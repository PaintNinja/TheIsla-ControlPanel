Accessing the console

Easy difficulty

In this tutorial, you'll learn how to access the console so that you can run any command as a top-rank user.

It's quite easy really, but just in case you're not sure, just keep reading... I'll tell you how to do it step by step. :)

  1. Open up The Isla CP (Control Panel)
  2. Click on "Console" under the server section
  3. Enter the password to login to the console

That's it! :D

How to apply a rank to a specific user

Medium difficulty

This has been tagged as "Medium difficulty" because it takes a while to get used to and if done wrong, it's not obvious unless you know the symptoms of PEX errors.

To apply a rank to a specific user, use the following syntax: pex user USERNAME group set RANK

Note: USERNAME and RANK are cAsE-SeNsItIvE, so the rank legend is completely different from Legend and the user Notch is completely different from notch.

Examples

  • pex user Notch group set VIP
  • pex user LOFTYBOY group set Moderator
  • pex user LOFTYBOY group set Co-Owner
  • pex user TJBennerzz group set Member
  • pex user Kasanii group set CM

Dealing with duped/hacked in-game money

Incomplete

I wasn't really sure whether this should go in the Console, Seamless maintenance or in-game economy management, so I decided to put it here and mention all the methods of dealing with hacked or duped in-game money.

Obviously you want to make sure the server's as moderated as possible, but there's times when staff just 'aint enough - like when you gotta sort someone out who has a ridiculous amount of in-game money in their /balance. This is where you come in! Only Admins or higher are allowed to manage the server's bank, as this is not as simple as just remove the money and you're done. You need to know the root cause of the issue, as well as sorting out the issues caused by the root at the same time.


Find potentially duped/hacked in-game money

Medium difficulty

There are various methods of doing this...

  1. In-game
  2. Console
  3. Wizard - in-game
  4. Wizard - console

Method 1: In-game

This method is fairly easy - as long as you are able to join the server for a bit without getting asked for help or needing to sort out other stuff - which is very unlikely.

Anyway, here's how step-by-step for this method:

  1. Start CraftMine
  2. Join the server
  3. Type /baltop
  4. Look for users with a suspiciously high amount of money
    • If someone has been on the server for a very long time then they are allowed to have loads of money - with exceptions - of course
    • If someone is fairly new to the server then anything over £3k is suspicious
    • If someone has only been welcomed to the server just a couple of hours ago, anything over £500 is suspicious
  • If there's something suspicious, carry on to the next section of this tutorial
  • If there's nothing suspicious, then you don't need to take any action and you're all up-to-date! :)

Method 2: Console

This method is similar to method 1, except that you won't need to help anyone or sort out other stuff. This is the preferred method, unless you are already in-game - where method 1 would be preferred if that is the case.

Here's how step-by-step for method 2:

  1. Open up The Isla CP (Control Panel)
  2. Click on "Console" under the server section
  3. Enter the password to login to the console
  4. Type /baltop
  5. Look for users with a suspiciously high amount of money
    • If someone has been on the server for a very long time then they are allowed to have loads of money - with exceptions - of course
    • If someone is fairly new to the server then anything over £3k is suspicious
    • If someone has only been welcomed to the server just a couple of hours ago, anything over £500 is suspicious
  • If there's something suspicious, carry on to the next section of this tutorial
  • If there's nothing suspicious, then you don't need to take any action and you're all up-to-date! :)
Yeah I know, there's more steps on this one than the first method, but this is much quicker than waiting for the game to load, I can guarantee you that!

Method 3: Wizard - in-game

This is really the same as method 1 - but guided.

  1. Open up The Isla CP (Control Panel)
  2. Hover over "Guides" in the menu on the left
  3. Click on "Find potentially duped/hacked in-game money - in-game" under the "Wizard" section on the left menu
  4. Follow the guided steps

Method 4: Wizard - console

This is really the same as method 2 - but guided.

  1. Open up The Isla CP (Control Panel)
  2. Click on "Find potentially duped/hacked in-game money - console" under the "Wizard" section on the left menu
  3. Follow the guided steps

Grave cleanup

As you may have noticed, you rarely see graves at spawn, the market, etc... if ever! - That's because I go around almost daily maintaining the server's services by destroying any graves in specific areas. This keeps the place looking new, clean and welcoming rather than cluttered with loads of old graves all over the place.

In this tutorial, I'll show you how to do it! :D


Knowing where to clean up and where not to

Easy difficulty

This is very important, because the last thing you want is players complaining that their graves have suddenly gone when they were going to get their stuff back later.

Where you should cleanup
  • Anywhere at /spawn
  • In the safezone areas at /warp wild
Where you should leave alone
  • Random places in the wilderness
  • Faction land (apart from Safezone and Warzone)

How to cleanup the graves

Easy difficulty

It's quite simple really, but here's how just incase you're unsure...

  1. Go to the grave you want to remove
  2. Make sure your inventory is empty
  3. Type /gm c to change to creative mode
  4. Hit the grave
  5. Type /ci
  6. Type /gm s to change back to survival mode

That's it! :)
