Donation management

Applying donation rewards

In this tutorial, you'll learn how to apply donation rewards to players who have donated.

Note: This only applies to ranks, rank upgrades and the unban package. Everything else in the donation store is automated. :)

Firstly, you can only apply donation rewards if you have been notified that the player has donated through Enjin, with the exception of event prizes with my permission.


Knowing when someone has donated

Medium difficulty

There are two ways to do this:

  1. Going to the server's website and checking your inbox
  2. Using the guided mode in The Isla CP (Control Panel)

I'll explain method 1 and then method 2...

Method 1

  1. Open up The Isla CP (Control Panel)
  2. Click on "Inbox" under the community section
  3. Look for a recent unread message with the subject "Purchase notification"
    • If there is no recent unread message with that subject, you are up to date and don't need to apply a donation reward
    • If there is a recent unread message with that subject, open it up
  4. Check to see if the message is already replied to by me, indicating that I've already applied the reward
    • If already applied, you don't need to apply the donation reward again

That's method one for you... Just open up your Enjin inbox, check if there's a notification and check if I've already done it, simple when I just say it like that, right?

Method 2

This way's basically the same as method one, but... you're guided whilst doing it. Handy if you forgot how to do it but don't want to read a long and boring tutorial like this one, for example. ;)

  1. Open up The Isla CP (Control Panel)
  2. Hover over "Guides" in the menu on the left
  3. Click on "Check if someone has donated"
  4. Follow the guided steps

Sure, this may be easier but it's a lot slower than the first method. Once you're used to method one you'll find that it's miles quicker than this method. But still, this can still help if you're new to the process or forgot it. :)

Applying the donation rewards

Easy-Medium difficulty

Some of the donation rewards are automatically applied as soon as someone donates, with them also being removed when the donation expires (if it can expire). Examples of this are forum tags and access to donator-only forum categories, like the VIP forum category to name one.

Unfortunately, the same doesn't apply for PermissionsEX ranks, as there seems to be a bug in CraftMine or PEX (PermissionsEX) that prevents Enjin from setting ranks automatically, so we do this manually instead...

There's two ways to do this:

  1. Using the console and control panel normally
  2. Using the guided mode in the control panel
Either way's fine, but as I explained before, the first method is the quickest once you're used to it, and the second method is best when you forgot how to do it or when you're still learning how to do it. or... if you're lazy. :P

Method 1

Tip! Refer to the "Console" tutorial section for detailed help on setting ranks.
  1. Open up The Isla CP (Control Panel)
  2. Click on "Console" under the server section
  3. Enter the password to login to the console
  4. Set the appropriate rank
    • VIP: pex user USERNAMEHERE group set VIP
    • Legend: pex user USERNAMEHERE group set Legend
  5. Let the user know that they have recieved what they paid for
    • mail send USERNAMEHERE Your perks have been fully applied. Enjoy! :D

Method 2

  1. Open up The Isla CP (Control Panel)
  2. Hover over "Guides" in the menu on the left
  3. Click on "Apply donation reward"
  4. Follow the guided steps

Congrats! You've made it to the end of this tutorial. Now you'll be able to professionally provide the appropriate rewards to users when they donate. :)
